Microservices Core Architecture (Java 25 + Spring Boot 4.0.5)

A scalable and modular microservice architecture built with Java 25 and Spring Boot 4.0.5, using Maven dependency. The project includes a shared common module for reusable components, a dedicated database module for data layer isolation, and secure authentication via OAuth 2.0 using Spring Authorization Server. Authorization is implemented using a policy-based access control model with JCasbin for flexible and fine-grained permissions.
